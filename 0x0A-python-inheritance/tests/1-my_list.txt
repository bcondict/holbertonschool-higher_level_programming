1-my_list
=========

    >>> MyList = __import__('1-my_list').MyList

adding elements of a list:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(3)
    >>> my_list.append(6)
    >>> my_list.append(4)
    >>> print(my_list)
    [1, 3, 6, 4]
    >>> my_list.print_sorted()
    [1, 3, 4, 6]

adding negative elements:
    >>> my_list = MyList()
    >>> my_list.append(-1)
    >>> my_list.append(-5)
    >>> my_list.append(-7)
    >>> my_list.append(-2)
    >>> print(my_list)
    [-1, -5, -7, -2]
    >>> my_list.print_sorted()
    [-7, -5, -2, -1]

passing empty list:
    >>> my_list = MyList()
    >>> print(my_list)
    []
    >>> my_list.print_sorted()
    []
